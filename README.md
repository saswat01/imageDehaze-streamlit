# Haze removal web application

This streamlit app was built for haze removal of images on the web with option to tune the extent of dehazing. 

This work is an extension of [DOI:10.1007/s11554-017-0698-z](https://doi.org/10.1007/s11554-017-0698-z)

Use this link to access the web app: [webapp](https://image-dehazing.herokuapp.com/)
